L-Spiel

Ein (HTML-)-Brett-Spiel entwickelt auf einer
mehrstündigen Bahnfahrt von Janek und Patrick.
Erweiterungen und Verbesserungen erfolgten
in den Tagen darauf.

Entwicklung:

- Entwickeln/Testen mit index_devel.html (jsx wird im Browser
  umgewandelt)
- Code prüfen: "node test" lässt eslint laufen
- Compilieren: "node compile.js" erzeugt index.html und bundle.js
